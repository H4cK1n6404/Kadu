## Description

Kadu is a dynamically evolving instant messenger compatible with the
Gadu-Gadu and XMPP protocol. It can be run on all platforms supported
by the Qt toolkit.

Kadu is licenced under GPLv2 or newer.

http://www.kadu.im/
